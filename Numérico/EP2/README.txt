Para executar corretamente o código .py,
você terá que ter as versões especifica-
das abaixo para cada uma das bibliotecas

matplotlib==3.4.2
numpy==1.21.0
opencv-python==4.5.2.54
pandas==1.2.5
seaborn==0.11.1

Caso não tenha essas versões você pode
colar o código .py em uma célula de
jupyter notebook dentro do Google Colab,
outra forma também é fazer o upload do
arquivo .ipynb também no Google Colab.

Observe que este EP é a continuação do 
primeiro para executar as novas funções
você deve garantir que as todos os arquivos
estejam na mesma pasta (ou que o path seja
passado completamente).

Leia atentamente as instruções durante a
execução, para parar aperte o botão X do 
navegador ou de sua IDE, ou pressione o 0
quando no menu inicial
