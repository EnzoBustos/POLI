Para executar corretamente o código .py,
você terá que ter as versões especifica-
das abaixo para cada uma das bibliotecas

matplotlib==3.4.2
numpy==1.21.0
opencv-python==4.5.2.54
pandas==1.2.5
seaborn==0.11.1

Caso não tenha essas versões você pode co-
lar o código .py em uma célula de jupyter
notebook dentro do Google Colab, outra forma
também é fazer o upload do arquivo .ipynb tam-
bém no Google Colab
